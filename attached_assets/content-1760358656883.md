#### What is ICT OTE?

Optimal trade entry is basically a trading model based on the retracement of price in a trendy market.

Fibonacci retracement levels defined by the Inner Circle Trader are key to trade ICT OTE and are explained later.

When price moves in a trend either bullish or bearish, it does not go vertically straight rather it retrace and then rallies.

So, ICT OTE can be used for joining the trends at retracement levels and for the reversals at the key PD-Arrays of higher time frame.

**ICT Fibonacci Levels for OTE Setup**

To use ICT optimal trade entry pattern you should set your fibonacci retracement levels to the following as used by ICT.

|     |     |
| --- | --- |
| 0 | First Profit Scale |
| 0.5 | Equilibrium |
| 0.62 | OTE level 1 |
| 0.705 | 0TE level 2 |
| 0.79 | Optimal Trade Entry |
| 1 | 100% retracement level (Starting Position) |
| -0.5 | Target 1 |
| -1 | Target 2 |
| -2 | Symmetrical Price |
